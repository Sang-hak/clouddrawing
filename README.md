# 프로젝트 소개
### 평범한 일상을 특별한 일상으로, 구름그린
- 공간에 담긴 추억을 일기 형태로 남길 수 있어요
- 향후 일기를 썼던 곳을 다시 찾으면, 추억을 회상할 수 있게 알려드려요
